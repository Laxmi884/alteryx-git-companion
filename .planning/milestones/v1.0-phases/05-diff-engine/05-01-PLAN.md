    @dataclass(frozen=True, kw_only=True)   # slots=True removed â€” required for @property
    class DiffResult:
        """Complete diff result between two WorkflowDoc instances."""
        added_nodes: tuple[AlteryxNode, ...] = field(default_factory=tuple)
        removed_nodes: tuple[AlteryxNode, ...] = field(default_factory=tuple)
        modified_nodes: tuple[NodeDiff, ...] = field(default_factory=tuple)
        edge_diffs: tuple[EdgeDiff, ...] = field(default_factory=tuple)

        @property
        def is_empty(self) -> bool:
            """True when no additions, removals, modifications, or edge diffs."""
            return (
                not self.added_nodes
                and not self.removed_nodes
                and not self.modified_nodes
                and not self.edge_diffs
            )

    Public function signature:
    ```python
    def diff(
        match_result: MatchResult,
        old_connections: tuple[AlteryxConnection, ...],
        new_connections: tuple[AlteryxConnection, ...],
    ) -> DiffResult:
    ```

    Implementation structure:
    - `added_nodes`: `tuple(n.source for n in match_result.added)`
    - `removed_nodes`: `tuple(n.source for n in match_result.removed)`
    - `modified_nodes`: iterate `match_result.matched`, call `_diff_node(old.source, new.source)` only when `old.config_hash != new.config_hash` (fast-path skip for identical configs)
    - `edge_diffs`: call `_diff_edges(old_connections, new_connections)`

    `_diff_node(old_node, new_node) -> NodeDiff` â€” field-level config diff:
    - Use `DeepDiff(old_node.config, new_node.config, ignore_order=False, verbose_level=2)`
    - Build `field_diffs: dict[str, tuple[Any, Any]]` by processing DeepDiff output:
      - `values_changed`: `field_diffs[key] = (change["old_value"], change["new_value"])`
      - `dictionary_item_added`: `field_diffs[key] = (None, _get_nested_value(new_node.config, path))`
      - `dictionary_item_removed`: `field_diffs[key] = (_get_nested_value(old_node.config, path), None)`
      - `iterable_item_added` / `iterable_item_removed`: surface the PARENT list as atomic before/after â€” strip the trailing `[N]` index from the path and use the full list value as before/after. This satisfies CONTEXT.md: "list fields are atomic values â€” if any element differs, the whole list is before/after"
      - `type_changes`: `field_diffs[key] = (change.get("old_value"), change.get("new_value"))`
    - Filter out paths in `_EXCLUDED_FIELDS` frozenset (start empty per research recommendation â€” `frozenset()`)
    - If `field_diffs` is empty after processing, raise `ValueError` with message identifying `old_node.tool_id` â€” this is a developer bug (config_hash differed but DeepDiff found nothing after exclusions)
    - Return `NodeDiff(tool_id=old_node.tool_id, old_node=old_node, new_node=new_node, field_diffs=field_diffs)`

    Helper `_deepdiff_path_to_dotted(path: str) -> str`:
    - Converts `"root['key1']['key2']"` â†’ `"key1.key2"`
    - Strip `root` prefix, split on `']['`, strip quotes and brackets from each part
    - For list index paths like `"root['Fields'][3]"`, strip the numeric index (return parent path `"Fields"`) â€” this supports the atomic list treatment

    Helper `_get_nested_value(config: dict[str, Any], deepdiff_path: str) -> Any`:
    - Navigate the config dict following the DeepDiff path to retrieve the actual value
    - Used for `dictionary_item_added` and `dictionary_item_removed` to get the present side's value

    `_diff_edges(old_connections, new_connections) -> tuple[EdgeDiff, ...]`:
    - `old_set = frozenset(old_connections)`, `new_set = frozenset(new_connections)`
    - `removed = old_set - new_set`, `added = new_set - old_set`
    - Sort removed connections by `(src_tool, src_anchor, dst_tool, dst_anchor)` for deterministic output
    - Sort added connections the same way
    - Create `EdgeDiff(src_tool=..., src_anchor=..., dst_tool=..., dst_anchor=..., change_type="removed")` for each removed
    - Create `EdgeDiff(..., change_type="added")` for each added
    - Return `tuple(edge_diffs)`

    All imports from `alteryx_diff.models` (not sub-modules) per project convention.
