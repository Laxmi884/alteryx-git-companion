    `build_from_workflow(doc: WorkflowDoc) -> dict[str, Any]`:
    - `workflow_name` = `Path(doc.filepath).stem` (per Pitfall 2 — NO `.name` field)
    - `tool_count` = `len(doc.nodes)`
    - `tools` = list of `{"tool_id": int(node.tool_id), "tool_type": node.tool_type, "config": node.config}` for each node (per D-06, full config dict; per Pitfall 3, explicit `int()` conversion)

    - Build `nx.DiGraph` from doc.nodes and doc.connections (reuse pattern from `_graph_builder.py`: `int(node.tool_id)` for nodes, `int(conn.src_tool/dst_tool)` for edges)
